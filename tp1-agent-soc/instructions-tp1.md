# Instructions pour le TP1 — Agents et cybersécurité

## Étape 1 — Identifier les sources
Lister toutes les sources consultées par l’agent

## Étape 2 — Classer les sources par niveau de confiance
Classer les sources en fonction de leur fiabilité, en justifiant leur classement

## Étape 3 — Identifier l’attaque
Déterminer le type d’attaque en la nommant et en expliquant les caractéristiques qui permettent de l’identifier
- D'où vient l’attaque ?
- Quels sont les objectifs de l’attaquant ?
- Quels sont les impacts potentiels de l’attaque ?

## Étape 4 — Produire une réponse sûre de l’agent
Rédiger une réponse que l’agent pourrait fournir
